Installation
============

To install this module, you need to:

#. Go to apps
#. Look for mrp_bom_structure_xlsx module
#. Click install

Usage
=====

To use this module, you need to:

Go to the Bill of Materials form or list views, press 'Print > Export BoM
Structure to Excel'.
